Never edit or overwrite existing email files in created/emails/.

Always write new emails as a new .txt file with a new dated filename.

**Why:** User copies emails directly into Outlook. Editing an existing file risks corrupting a sent or in-progress draft.

**How to apply:** Any time an email needs to be written or rewritten, create a new file — e.g. jen-tracking-feed-collated-2026-04-15.txt. Leave the old file untouched.
