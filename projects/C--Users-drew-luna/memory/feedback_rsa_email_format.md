When producing RSA ad copy approval emails for any client, use this exact format for each ad block:

Campaign Name - Ad Group Name - Ad Name

[15 headlines, one per line, no labels]
[4 descriptions, one per line, no labels]
https://final-url

No blank lines between headlines, descriptions, and URL — all run together as one block. No blank line after the header line — the first headline runs immediately below it.

**Why:** User copies directly into Excel/Google Ads Editor. Labels and blank lines within the block break the paste flow.

**How to apply:** Replace all "Ad X - URL: https://..." headers and "Description X:" labels with this structure across every RSA approval email, for every client.

Brand - Brand - Ad 1
