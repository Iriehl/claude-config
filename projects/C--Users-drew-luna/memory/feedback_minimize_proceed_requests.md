Don't ask for approval between phases of a multi-step analysis or task. Once the overall plan is approved, work through steps sequentially and report findings without gating on confirmation at each step.

**Why:** User finds these check-ins disruptive and unnecessary. They want to see results, not be asked permission to continue.

**How to apply:** After plan approval, execute each phase in sequence. Report what was found at the end of each phase, then move to the next. Only pause if a decision point arises that genuinely changes the direction (e.g., data doesn't exist, a finding invalidates the approach).
